# Reflection

## What I Learned
I learned how to build a dynamic form that is generated from a JSON schema instead of hardcoding form fields. I also learned how validation rules and conditional rendering improve the user experience and make applications more flexible.

## Challenges
The main challenge was understanding how to structure the JSON schema and connect it with React components for rendering and validation.

## Mistakes
Initially, I thought hardcoding the form would be easier, but I realized that a JSON-driven approach is more scalable and easier to maintain.

## Time Spent
- Requirement analysis: 30 minutes
- Designing the solution: 45 minutes
- Implementation: 2 hours
- Testing: 45 minutes
- Documentation: 30 minutes

**Total Time:** Approximately 4.5 hours.

## Conclusion
This project improved my understanding of React, JSON-based form generation, validation, conditional rendering, and software architecture. It also showed me the importance of writing clear technical documentation.
