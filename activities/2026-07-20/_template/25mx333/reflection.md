# Reflection

This project gave me practical experience in implementing a machine learning algorithm without depending on external libraries like Scikit-learn. Instead of using a built-in function, I understood how K-Means works internally through centroid initialization, distance calculation, cluster assignment, and centroid updates.

One challenge I faced was understanding how the algorithm repeatedly updates the centroids until convergence. I also had to carefully handle the distance calculations and ensure that every data point was assigned to the correct cluster.

During the implementation, I initially found it difficult to decide when the algorithm should stop iterating. By adding a convergence condition based on centroid movement, I was able to make the solution more efficient and accurate.

I spent around 40 minutes understanding the requirements, 1 hour designing the logic, 2 hours implementing the algorithm, 30 minutes testing the results, and 20 minutes preparing the documentation.

Overall, this task improved my understanding of unsupervised learning, data clustering, iterative algorithms, and performance analysis. It also strengthened my problem-solving skills and confidence in implementing algorithms from scratch.
